Tools Available¶

expfactory-python¶

A python module for using these tools, including functions to compile psiturk-experiments. into a psiturk battery. and deploy in a virtual machine.

expfactory-battery¶

An instance of a psiturk experiment that contains a specific set of experiments, chosen by the user.

expfactory-experiments¶

The experiments deployed into the battery.
